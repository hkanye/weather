# 天气应用

## 简介

使用create-react-app、express开发的项目

调用免费的openweathermap接口获取数据

展示了搜索的城市的天气信息



## 相关技术

使用lerna管理的monorepo项目

包含3个packages

- pc端页面
- server服务端
- shared公共逻辑

pc包使用create-react-app初始化、开发，antd作为ui基础包

server端使用express



## 如何使用

### 安装

项目根目录运行脚本

```bash
npm install
```

### 运行

项目根目录运行脚本

```bash
npm start
```
